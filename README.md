# Agent Hendrix

A [Claude Code](https://code.claude.com) plugin marketplace for agent-assisted development.

## Marketplace

```bash
claude plugin marketplace add expki/agent-hendrix
```

## Plugins

| Plugin | Download |
| --- | --- |
| [**webgpu-lsp**](./webgpu-lsp) | `claude plugin install webgpu-lsp@agent-hendrix` |

See each plugin's linked README for setup and details.
